# 簡易版matrix_math_toolテスト仕様書

## 1. テスト環境

* コンパイラ：gcc
* OS：ubuntu

## 2. テスト項目

| ID | テスト項目 | 期待される結果 | 実行結果 | 判定 | 備考 |
| --- | --- | --- | --- | --- | --- |
| **T-01** | 各関数におけるNULLチェック検証 | 各関数にてメッセージ表示後 `exit 1` で終了 |  |  |  |
| **T-02** | 行列サイズ整合性 | メッセージ表示後 `exit 1` で終了 |  |  |  |
| **T-03** | 演算確認結果チェック | 演算結果が期待通りであること |  |  |  |
| **T-04** | メモリ解放確認 | Valgrindでリークが検出されないこと |  |  |  |

## 3. 異常系テスト

* **生成時のバリデーション**: `rows`または`cols`に `0` 以下の値を指定した場合に `exit 1` となること
* **メモリ確保失敗時**: `Matrix`型の変数`m`がNULLだった場合、適切にエラーを表示し `exit 1` となること

## 4. 共通項目
- `exit(1)`を使う際は直前にメッセージを表示させ`$?`コマンドを使用して確認

## 5. テスト実行時における使用項目
* **使用配列**: `create_matrix(5, 6)`および`create_matrix(6, 5)`
* **エラーメッセージ**: `fprintf(stderr,"{処理内容を記載}時にエラーが発生。処理を強制終了します")`
* **mallocの失敗方法**: create_matrixで使用するポインタ`m`に直接NULLを代入
