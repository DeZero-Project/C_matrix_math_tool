# 要件定義書：行列計算ツール

## 1. 目的
* 外部ライブラリに依存せず、C言語標準ライブラリのみを用いて基本的な行列演算を行うこと
* DeZero作成時の補助ツールとして利用
* Cを使った実践的なツール開発とその学習

## 2. 機能要件
- [ ] 行列の入力（手入力 / ファイル入力など）
- [ ] 基本演算（加算、減算、乗算）
- [ ] 単項演算（転置、スカラー倍）
- [ ] 特殊演算（平方根 `sqrt` を用いたノルム計算、行列式など）
- [ ] 計算結果の表示および出力
- [ ] 計算結果をファイルに出力

## 3. 非機能要件
- 依存性：C言語標準ライブラリのみ（stdio.h, stdlib.h, math.h 等）
- 実行環境：標準的なCコンパイラ（GCC/Clang等）でビルド可能であること
- 実行環境： 移植性を考慮した実装を行うこと(linux, windows双方で利用可能が条件)
- メモリ管理：動的確保を行い、使用後は適切に解放すること
- メモリ管理： カスタムメモリアロケータを実装してメモリ管理を行うこと

## 4. 制約事項
- インストールが必要な外部ライブラリは使用しない。
