# 設計書：行列計算ツール（仮）

## 1. データ構造
行列を管理するための構造体、またはポインタの管理方法を定義。
- 例：`typedef struct { int rows; int cols; double *data; } Matrix;`

## 2. モジュール構成
- `main.c`: ユーザーインターフェース、全体の制御
- `matrix_ops.c`: 行列計算のロジック（加減乗除など）
- `memory_mgmt.c`: メモリ確保・解放の管理

## 3. 主要アルゴリズム
- 行列乗算：3重ループによる実装
- メモリ確保：`malloc` を用いた1次元配列による2次元的アクセス

## 4. インターフェース（関数定義）
- `Matrix* create_matrix(int rows, int cols);`
- `void free_matrix(Matrix *m);`
- `Matrix* multiply_matrices(Matrix *a, Matrix *b);`
